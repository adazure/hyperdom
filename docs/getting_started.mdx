# Getting Started

## install

### New project

Create a skeleton appilcation with express backend:

```bash
npx create-hyperdom-app my-app

// or using yarn

yarn create hyperdom-app my-app
```

### Add to existing project


```javacscript
npm install hyperdom

// or using yarn

yarn add hyperdom
```

Use with browserify or webpack:

```javascript
const hyperdom = require('hyperdom')
```
